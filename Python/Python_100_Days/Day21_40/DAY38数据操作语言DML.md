# DML
### 定义
`DML`可以帮助将数据插入到二维表（`insert`操作）、从二维表删除数据（`delete`操作）以及更新二维表的数据（`update`操作）
### 功能
（1）将数据插入到二维表（2）从二维表删除数据（3）更新二维表的数据
### 任务
操作数据

### 1. `INSERT`插入
```sql
-- 依次插入所有完整行
INSERT INTO `table_name` VALUES ('值1', '...', '值n')

-- 指定行插入
INSERT INTO `table_name` (col_name) VALUES ('值1')

-- 批量插入
INSERT INTO `table_name`
    ('col_name', `col_intro`)
VALUES
    ('外国语学院', '学习歪果仁的语言的学院'),
    ('经济管理学院', '经世济民，治理国家；管理科学，兴国之道'),
    ('体育学院', '发展体育运动，增强人民体质');

-- 替换插入
-- 将another_table_name表中a列和b列的所有数据，按顺序批量插入到table_name表的col_name和col_intro字段中
INSERT INTO `table_name`
    ('col_name', `col_intro`)
SELECT `a`, `b`
    FROM `another_table_name`;
```

### 2. `delete`操作
```sql
-- 基础语法
DELETE FROM 表名
WHERE 条件;

-- 删除指定行
DELETE FROM table_name WHERE col_id=1;

-- 删除全表行
DELETE FROM table_name; -- 仅删除数据，保留表结构
```

### 3. UPDATE
```sql
-- 基础语法
UPDATE 表名
SET 字段1=值1, 字段2=值2, ...
WHERE 条件;

-- 修改单字段
UPDATE table_name SET `name`='李四' WHERE `id`=1;

-- 修改多字段
UPDATE table_name
SET `name`='李四', `gender`='男'
WHERE `id`=1;
```
