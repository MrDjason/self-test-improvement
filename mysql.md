# MySql

## 管理
- **登录MySql**

```mysql
mysql -u root -p
```

- **列出所有数据库**

```mysql
show databases;
```

- **选择要操作的数据库**

```mysql
use Database;
```

- **显式指定数据库的所有表**

```mysql
use databases;
show tables;
```

- **显示表中结构**

```mysql
desc tablename;
```

- **显示表中所有数据**

```mysql
select * from tablename;
```
