### 1.8 Linux终端命令格式  
**终端**  
终端是软件控制台，在终端中输入指令可以快速操作计算机  
终端打开方式：
- 1.右击桌面打开终端
- 2.快捷键：`ctrl + alt + t` 

**终端指令的格式：`command [-options] [parameter]`**  
说明：
- `command`：命令名，相应功能的英文单词或者单词缩写
- `[options]`：可用来对命令进行控制，也可省略
- `parameter`：传给命令的参数，可以是零个、一个、或多个  

`-`代表选项，`[]`代表可选

**查阅命令帮助信息**  
查看帮助有两种方式  
**1. man(manual)命令**
- 回车 一行
- 空格 一页
- f - 下一屏
- b - 上一屏
- q - 退出  

**2. 命令 --help**
```bash
command --help
```

### 1.9 [重点]常用命令-显示文件和目录
**目标**  
- `pwd` 显示当前所在路径
- `tree` 以树状列表的方式显示文件夹的内容
  - `tree` 当前显示当前目录下树状图
  - `tree aaa` 指定显示当前目录下aaa文件夹的树状图
- `ls` 以列表的方式显示文件夹的内容(ls为list简写)
- `ls -a`  显示当前目录所有文件(包含隐藏文件)
### 1.10 常用命令-显示文件和目录
隐藏文件，在linux中以`.`开头  
- `-l` 以详细信息的方式显示列表
- `-h` 以更加人性化的方式显示文件夹内容  
`ls -l`  
drwxr-xr-x 3 demo demo `4096` 11月  3 21:25 a  
`ls -l -h`  
drwxr-xr-x 3 demo demo `4.0K` 11月  3 21:25 a


```
-rw-r--r-- 1 demo demo  59 11月  3 22:05 1.txt
-rw-r--r-- 1 demo demo  31 11月  3 21:59 2.txt
drwxr-xr-x 3 demo demo 4096 11月  3 21:25 a
-rw-r--r-- 1 demo demo  24 11月  3 21:11 a1_copy.txt
-rw-r--r-- 1 demo demo  24 11月  3 21:04 a1.txt
-rw-r--r-- 1 demo demo  19 11月  3 21:04 a2.txt
drwxr-xr-x 2 demo demo 4096 11月  3 21:25 c
```
其中
- `-`在开头表示这是文件，`d`表示这是一个目录|链接文件
- `rwxr-xr-x`、`rw-r--r--`表示文件的权限
- `1`、`2`、`3`表示硬链接数,文件有几个“快捷方式”
- `demo` `demo`表示文件所属用户、文件所属小组
- `4096`、`24`等表示文件大小

### 1.11 [重点]常用命令-切换目录
**目标**  
- 能够使用cd切换`根目录`、`家目录`、`当前目录`和`上层目录`
- **`cd`**：change directory

**用法**  
cd后面可以跟绝对路径，也可以跟相对路径，cd后如果省略目录，则默认切换当前用户的`主（家）目录`
```
/  
├── bin  
├── etc  
└── home(上级)  
    └── teacher(当前目录)  
        ├── a  
        └── b  
```
**绝对路径**：`/home/teacher/a`  
**相对路径**：` ./a: teacher`

| 指令    | 说明         |
|---------|--------------|
| cd 路径 | 切换到指定路径 |
| cd      | 回到主目录   |
| cd ~    | 回到家目录   |
| cd .    | 当前目录     |
| cd ..   | 上级目录     |
| cd -    | 上次所在目录 |

### 1.12 [重点]常用命令-创建文件和目录
**目标**  
- `mkdir`创建目录
- `touch`创建文件
- `gedit`打开文件

**1. 创建目录：`mkdir`**  
通过`mkdir`命令可以创建一个新的目录  
`mkdir ./a`在当前目录下创建`a`文件夹 
|选项|含义|
|:---|:---|
|-p|递归创建目录|

如 `mkdir -p ./a/b/c/d` 当前目录创建`a/b/c/d`四个依次包含的文件夹

**2. 创建文件：`touch`**  
通过`touch`命令可以创建一个新的文件
```bash
touch hello.txt           # 创建1个文件
touch a.txt a1.txt a2.txt # 创建3个文件
```

**3. 打开文件：`gedit`**  

通过`gedit`命令打开单个文件，并可以编辑。编辑时终端处于等待状态  
```bash
gedit 1.txt             # 打开1个文件
gedit a.txt b.txt c.txt # 打开3个文件
```

### 1.13 [重点]常用命令-删除文件和目录
**目标**  
通过`rm`（remove）删除文件或者文件夹  

常用参数及含义：
|选项|含义|
|:---|:---|
|-i|以进行交互式方式执行|
|-f|强制删除，忽略不存在的文件，无需提示|
|-r|递归删除目录下的内容，删除文件夹（目录）时必须加此参数|

```
top/
└── a/
    └── b/
        └── c/
            └── d
```
`rm ./a`无法删除，`'./a'`是一个目录  
`rm -r a`可以删除

### 1.14 [重点]常用命令-文件拷贝
**目标**  
通过`cp`拷贝文件或者目录  

**用法**  
`cp 源文件路径 目录文件路径`

常用选项说明：  
|选项|含义|
|:----|:----|
|-a|该选项通常在复制目录时使用，它保留链接、文件属性，并递归地复制目录，简单而言，保持文件原有属性。|
|-f|覆盖已经存在的目标文件而不提示|
|-i|交互式赋值，在覆盖目标文件之前将给出提示要求目标确认|
|-r|若给出的源文件时目录文件，则cp将递归复制该目录下的所有子目录和文件，目标文件必须为一个目录名。|
|-v|显示拷贝进度|

### 1.17 常用命令-日历、日期指令
**目标**
- `cal` - 日历(calendar)
- `date` - 日期显示


**用法**
- `cal` 查看日历
  - `-3` 显示上月、当前月、下月
  - `-y` 显示一年日历
  - `-j` 以一年中第xx天的格式来显示日历
- `date` 显示日期
  - `date "+%Y%m%d%H%M%S%"` 查看当前年月日时分秒
  - `%F`等价于`+%Y-%m-%d` 如显示`2025-12-09`
  - `%T`等价于`%H:%M:%S™` 如显示`10:12:41`
  - `date +%F %T` 如显示`2025-12-09 10:12:41`

### 1.19 [重点]常用命令-文件查看cat
**目标**
- `cat` 查看或合并文件内容  

**1.查看文件**  
- cat 文件名
  - `-n` 查看文件的时候，对每一行进行编号
  - `-b` 非空行进行编号
  - `-s` 连续2行以上的空行，只显示1行  
  - 多个选项可以用一个`-`来连接，比如`cat -s -b 1.txt`可写作`cat -sb 1.txt`

**2.连接文件，把两个文件合并到一起查看输出**
- `cat` 文件名1 文件名2

### 1.20 [重点]常用命令-文件查看more
**目标**
- `more 文件名` - 分页查看文件内容  

**用法**
- `more 文件名`用来分屏(分页)查看文件内容
  - 显示之后`space`向下滚动
  - 显示之后`ctrl+b`向上滚动
  - `+num`从第`num`行开始查看文件
  - `-p`先清屏，再查看文件
  - `-s`连续`2`行以上的空行，只显示`1`行
  - `more 文件名1 文件名2` 把两个文件合并显示

**常用操作命令**
- `Enter`向下n行，需要定义，默认为1行
- `Ctrl+F`向下滚动一屏，F(front，前进)
- `Ctrl+B`返回上一屏，B(back，后退)
- `Space`向下滚动一屏幕
- `q`退出`more`
