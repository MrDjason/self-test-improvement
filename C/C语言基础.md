### 1.Helloworld原理分析

```c
#include <stdio.h>
int main()  // 主入口
{
    printf("Hello World");
    return 0;
}
```
- `#include` 预处理，提前找到需要用到的东西
- `stdio` StandardInputandOutput
- `int` 表示程序的结果是一个整数
- `main` 程序的主入口
- `{}` 范围
- `printf(" ")` 业务代码
- `return 0` 程序结束

程序通过`HelloWorld.c`编译成01的`HelloWorld.obj`文件，`HelloWorld.obj`再与`stdio.h`连接为`HelloWorld.exe`可执行文件，最后运行

### 8.9.10.变量

变量是存储数据的容器，程序运行时可以修改里面的值，经常发生改变的值可以定义为变量  

- 定义格式

```c
数据类型 变量名;
数据类型 变量名 = 初始值;
```

- 常用变量类型

| 数据类型 | 含义         | 示例值   | 输出格式符 |
| -------- | ------------ | -------- | ---------- |
| int      | 整型（整数） | 18, 100  | %d         |
| float    | 单精度浮点型 | 3.14, 6.6| %f         |
| char     | 单个字符     | 'A', '5' | %c         |

- 命名规则

1. 只能用 **字母、数字、下划线**  
2. 不能以 **数字开头**  
3. 区分大小写（age 和 Age 是两个变量）  
4. 不能用C语言关键字（如 int、main、return）  
5. 变量只能存一个值  
```c
int a = 10;
a = 20; // 覆盖原值，a现在为20
```
6. 变量名不允许重复定义  
```c
int a = 10;
// int a = 20; // 错误：重复定义变量a
```
7. 一条语句可以定义多个变量  
```c
int a = 10, b = 20, c = 30; // 同时定义3个int变量
```
8. 变量在使用前必须赋值
9. 变量有作用域范围

```c
#include <stdio.h>
int main()
{
    // 1. 定义变量并赋值
    int age = 18;
    float height = 175.5;
    char ch = 'A';

    // 2. 输出变量
    printf("年龄：%d\n", age);
    printf("身高：%f cm\n", height);
    printf("字符：%c\n", ch);

    return 0;
}
```

### 14.整数类型15.整数类型的两个扩展

| 类型         | 取值范围（32位/64位系统）                                                                 | 内存占用（字节） |
|--------------|------------------------------------------------------------------------------------------|------------------|
| `short`      | -32768 ~ 32767                                                                            | 2                |
| `int`        | -2147483648 ~ 2147483647（10位数）                                                         | 4                |
| `long`       | 32位：-2147483648 ~ 2147483647（10位数）<br>64位：-9223372036854775808 ~ 9223372036854775807（19位数） | 4 / 8            |
| `long long`  | -9223372036854775808 ~ 9223372036854775807（19位数）（C99标准）                             | 8                |

- **`short`**：短整型，占2字节，适合存储范围较小的整数。
- **`int`**：标准整型，占4字节，是日常开发中最常用的整数类型。
- **`long`**：长整型，在32位系统与`int`占用相同内存，64位系统下扩展为8字节，可存储更大范围整数。
- **`long long`**：超长整型，固定占8字节，用于存储超大范围整数。
- 所有整数类型均为**有符号数**，若需无符号版本，可在类型前添加`unsigned`关键字（如`unsigned int`），取值范围会变为非负数区间。

```c
#include <stdio.h>

int main()
{
    short a = 10;
    print("%d\n", a);
    int b = 100;
    print("%d\n", b);
    long c = 1000L;
    print("%ld\n", c);
    long long d = 10000LL;
    print("%lld\n", d)

    // 利用 sizeof测量每一种数据类型占用字节
    print("%zu\n", sizeof(short));
    print("%zu\n", sizeof(a));
    print("%zu\n", sizeof(b));
    print("%zu\n", sizeof(c));
    print("%zu\n", sizeof(d));
    return 0;
}
```

- 有符号整数 `signed`

```c
int a = -100; // signed int a = -100;
```


- 无符号整数 `unsigned`

```c
// 加入定义一个变量表示序号
unsigned int f = 999;
printf('%u', f) // 999

unsigned int f = -20; 
printf("%u", f) // 4294967276
```

### 4.隐式转换

计算时数据类型不一样不能直接运算，需要转成一样的，才能运算  

隐式转换：把一个取值范围小的，转成取值范围大的(例如二进制在int数前面补零)  

```c
int a = 10;
double b = 3.14;
double c = a + b;
```

```c
int a = 10;
double b = 20.0;
c = a + b
```

`short char`类型的数据在运算时先提升为int，再进行运算

```c
short a = 10;
short b = 20;
int c = a + b;
```

| 类别         | 具体内容                                                                 |
|--------------|--------------------------------------------------------------------------|
| 取值范围顺序 | `char < short < int < long < long long < float < double`                 |
| 触发时机     | 不同数据类型进行**计算、赋值**等操作时，会自动触发隐式类型转换             |
| 转换规则1    | 取值范围小的类型与取值范围大的类型运算时，**小范围类型先提升为大范围类型**，再参与运算 |
| 转换规则2    | `char`、`short` 类型在参与运算时，会**直接先提升为 int 类型**，再进行后续运算 |

### 5.强制转换

如果把取值范围大的，赋值给取值范围小的，就需要进行强制转换  

格式：**目标数据类型 变量名 = (目标数据类型)被强转的数据;**

```c
int b = 10;
short i = (short)b;
```

| 类别           | 具体内容                                                                 |
|----------------|--------------------------------------------------------------------------|
| 取值范围顺序   | `char < short < int < long < long long < float < double`                 |
| 触发时机       | 将**取值范围大的类型**赋值给**取值范围小的类型**时，需要使用强制类型转换 |
| 转换格式       | 在变量前添加 `(目标类型)`，例：`short i = (short)b;`                     |
| 强制转换弊端   | 可能导致数据精度丢失或数据错误                                           |

### 7.8.switch

switch作用：把所有的选择一一列举出来，根据不同的条件任选其一  

执行流程：
- 首先计算表达式的值
- 依次和case后面的值进行比较，有则执行，遇到break结束
- 都不匹配则执行default里面的语句，然后结束整个switch语句

```c
switch(表达式){
    case 值1:
        语句体1;
        break;
    case 值2:
        语句体2;
        break;
    ...
    default:
        语句体n;
        break;
}
```

```c
int week = 1;

switch (week)
{
    case 1:
        printf("1");
        break;
    case 2:
        printf("2");
        break;
    case 3:
        printf("3");
        break;
    default:
    printf("无匹配值");
}
```

| 序号   | 内容                                       |
|--------|--------------------------------------------|
| I      | 表达式：计算结果只能为（字符/整数）         |
| II-1   | case：值只能是（字符/整数）的字面量，不能是变量 |
| II-2   | case：值不允许重复                         |
| III    | break：表示中断，结束的意思，结束switch语句 |
| IV     | default：所有情况都不匹配，执行该处的内容   |

### 9.case的穿透

**case穿透的规则**
- 1.根据小括号中表达式的结果去匹配对应的case
- 2.执行对应case里面的代码
- 3.如果在执行过程中遇到了break，那么直接结束整个switch。没有遇到break就会执行下面case中的代码，直到遇到break，或者把整个switch中所有的代码全部执行完了才会结束

```c
{
case 1:
case 2:
case 12:
    printf("冬季");
    break;
case 3:
case 4:
cas3 5:
    printf("春季");
    break;
case 6:
case 7:
cas3 8:
    printf("夏季");
    break;
case 9:
case 10:
cas3 11:
    printf("秋季");
    break;
}
```

### 7.数组在内存中的深入剖析

## 指针

### 1.指针

指针变量定义格式  

`数据类型* 变量名 ： 要跟指向变量的类型保持一致  标记  自己起的名字`

**查询、存储数据**

格式
```c
// * 指针名
```

```c
{
int a = 10;
// 定义一个指针取指向变量a
int *p = &a; // 获取a的变量内存 地址并存储到指针中

// 利用指针去获取变量中的数据
printf("%d\n", *p);

// 利用指针取存储数据 / 修改数据
*p = 200;

// 输出打印
printf("%d\n", *p);
printf("%d\n", a);
}
```
其中  
`*p`是解引用运算符：通过内存地址获取数据  

- 指针变量占用的大小，跟数据类型无关，跟编译器有关
- 给指针变量赋值的时候不能把一个数值赋值给指针变量

指针 = 内存地址  
指针变量 = 存储着内存地址的变量  
指针可以利用`*（解引用运算符）`可以查询数据、存储数据

### 2.指针的第一个作用

- 作用1：操作其他函数中的变量  
- 作用2：函数返回多个值  
- 作用3：函数的结果和计算状态分开  
- 作用4：方便的操作数组和函数
